# ✦ AI Trader BOT v2.0 — Elite Edition

An automated quantitative trading system featuring a high-performance **Elite V2 Dashboard**, multi-indicator confluence engine, and Groq-powered AI sentiment analysis.

![Dashboard Preview](https://via.placeholder.com/1200x600?text=AI+Trader+BOT+Elite+V2+Dashboard)

## 🚀 Key Features

- **Elite V2 Command Center**: A professional-grade backtesting and monitoring dashboard with glassmorphic UI and 3D-style controls.
- **8-Indicator Confluence Engine**: Trades are only executed when your specialist team agrees. Includes RSI, MACD, EMA Cross, Supertrend, Bollinger Bands, VWAP, Mystic Pulse V2.0, and Candlestick Patterns.
- **AI Sentiment Brain**: Real-time news analysis powered by Groq (LLaMA-3) to filter out noise and confirm market direction.
- **Historical Backtest Engine**: Simulate years of trading data in seconds with detailed KPI tracking (ROI, Win Rate, Drawdown).
- **Alpaca Integration**: Seamless connection to Alpaca for both Paper and Live trading.

## 🛠️ Technical Stack

- **Backend**: Python 3.10+, FastAPI, Pandas, TA-Lib, Groq SDK.
- **Frontend**: HTML5, Vanilla CSS3 (Custom Glassmorphism), Javascript (ES6+), TailwindCSS (CDN).
- **Broker**: Alpaca Markets API.
- **Data**: Yahoo Finance (Historical) & TradingView (Real-time).

## ⚙️ Quick Start

1. **Clone the repository**:
   ```bash
   git clone https://github.com/YOUR_USERNAME/AI-Trade-Bot.git
   cd AI-Trade-Bot
   ```

2. **Setup Backend**:
   ```bash
   cd backend
   pip install -r requirements.txt
   cp .env.example .env # Add your API keys
   python main.py
   ```

3. **Launch Dashboard**:
   Open `frontend/index.html` in your browser.
